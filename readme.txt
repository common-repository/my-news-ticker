== Description ==
This is news ticker wordpress plugin. By using a simple [tickr_list] shortcode easily can use news ticker in your site. In this plugin easily control scroll time and can use multiple ticker. And also scroll news from everywhere by using post_type. 
If you need any customize the news sticker contact with me via skype: abujahed19



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After Activated the plugin use [tickr_list] shortcode where you want to use the news ticker.

Shortcodes

<strong>Use news ticker shortcode</strong>
<pre>[tickr_list]</pre>

<strong>Here can change</strong>
<pre>[tickr_list category="" count=""]</pre>


== Frequently Asked Questions ==
= To change everything easily used  count="" post_type="" category="" id="" time="" shortcode attribute. 
How can can change news scrolling time?
Here use attribute time in shortcode default scroll time is 1. if want increase the time can use 15 or if want reduce time can use 05.
